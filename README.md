# ringbuf-jk - 工业级环形缓冲区

由 **星珩软件工作室 J.K** 开发并贡献给自由软件社区

## 介绍
- 无锁单读单写
- 完美支持 uint32_t 计数溢出回绕
- 正确判断缓冲区满/空/覆盖
- 自带帧头 0x55 + 帧尾 0xAA 校验
- 适用于：STM32、单片机、串口、传感器、驱动开发

## 修复的核心 BUG
1. 修复 WW < RW 错误判断
2. 修复溢出覆盖逻辑错误
3. 支持 WW=1, RW=1000, BUF_LEN=1024 → 未读=25 正确计算
4. 真正工业级稳定逻辑

## 使用方法
直接复制代码到工程即可使用
